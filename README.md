
# Sistema Nacional de Assistência Jurídica Inteligente (SNAJI)

Arquitectura institucional .gov para assistência jurídica inteligente,
auditável e anti-alucinação.

## Stack
- FastAPI
- React + TypeScript
- PostgreSQL + pgvector
- Docker
- Multi-agent legal orchestration
- Hybrid RAG

frontend/
infra/
